### Build a RESTful API using Node.js, Express.js and Mongodb that allows users to CRUD user profiles.

1. The API should have authentication and authorisation using JWT Tokens.
2. Implement pagination and sorting for the user profile endpoints.
3. Write unit test for each endpoint.

### Instructions:

1. Code can be written in any standard IDE.
2. Code to be written in TDD (Test Driven Development) approach.
3. Standard error handling should me demonstrated.
4. Code should not be copied or sourced from any third party website.
5. Code to be hosted in personal git repo and link to be provided to downlaod the code.
